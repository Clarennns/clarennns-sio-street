# SIO Street 👕

SIO Street est une application web de gestion de boutique de vêtements streetwear réalisée dans le cadre du BTS SIO.

## 🚀 Fonctionnalités

### Côté Client
- Consultation du catalogue de vêtements.
- Filtrage par catégories (T-shirts, Sweats, Accessoires).
- Gestion du panier d'achat.
- Inscription et connexion sécurisée.
- Passage de commande et suivi.

### Côté Administration
- Gestion du catalogue (Ajout, modification, suppression d'articles).
- Suivi des commandes clients.
- Gestion des stocks en temps réel.

## 🛠️ Technologies utilisées

- **Frontend :** HTML5, CSS3 (Bootstrap ou Tailwind), JavaScript.
- **Backend :** PHP (Architecture MVC) ou Framework (Laravel/Symfony).
- **Base de données :** MySQL.
- **Outils :** Git, Composer.

## 📂 Structure du projet

- `/assets` : Images, CSS, JS.
- `/controllers` : Logique de traitement.
- `/models` : Interactions avec la base de données.
- `/views` : Fichiers d'affichage (Templates).
- `/config` : Configuration de la base de données.
